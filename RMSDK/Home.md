# RMCCore Library for Windows, Mac , iOS and Android #

RMCCore Library has C++ API which can be used by Windows, Mac, iOS and Android programmer. It is design for accessing NXL file, parsing Adhoc and Central policy from RMS ([NextLabs Rights Management Server](https://bitbucket.org/nxtlbs-devops/rightsmanagement-wiki/wiki/RMS/NextLabs%20Rights%20Management%20Server)) online services.
