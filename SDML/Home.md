# SkyDRM Mac Library API Wiki #

SDML API is designed for Mac programmer to develop user mode application by accessing RMS ([NextLabs Rights Management Server](https://bitbucket.org/nxtlbs-devops/rightsmanagement-wiki/wiki/RMS/NextLabs%20Rights%20Management%20Server)) online services and manageing the local NXL files.
